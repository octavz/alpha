# Alpha Shared Package

Shared TypeScript code between backend, web, and mobile.

## Features

- Shared TypeScript interfaces
- API client configurations
- Validation schemas (Zod)
- Utility functions
- Constants

## Usage

This package is used as a dependency by:
- Backend package
- Web frontend package  
- Mobile frontend package

## Building

```bash
bun run build
```

## Development

- `bun run build` - Build TypeScript
- `bun run watch` - Watch mode for development
